# Project Concept

## Why (One Sentence)

Qor-logic is the canonical source of truth for all S.H.I.E.L.D. governance skills — ingesting raw skills from any trustworthy source, processing them into compliant formats, and compiling them for different LLM agent runtimes to enable end-to-end autonomous building with governance confidence.

## Vibe (Three Keywords)

1. **Canonical** — one source of truth, no duplicates, no drift between agent formats
2. **Fluid** — skills flow from raw dump → processed S.H.I.E.L.D. → compiled per-LLM format
3. **Autonomous** — if brainstorming is good enough, the build process runs with minimal human intervention

## Anti-Goals (What This Is NOT)

- **Not an agent runtime** — skills are prompts/instructions, not executable code
- **Not tied to one LLM** — outputs to Claude, Gemini, Copilot, Cursor, Kilocode formats
- **Not a replacement for FailSafe** — this is the skill engine; FailSafe is the governance engine
- **Not a chatbot** — skills are structured governance protocols, not conversational templates
